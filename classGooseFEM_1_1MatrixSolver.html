Detailed Description

template<class Solver = Eigen::SimplicialLDLT<Eigen::SparseMatrix<double>>>
class GooseFEM::MatrixSolver< Solver >

Solver for Matrix().

The idea is that this solver class can be used to solve for multiple right-hand-sides using one factorisation.
